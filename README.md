# 开发日报

A CLI tool for generating development daily reports.

## 使用说明

本工具用于记录和管理每日的开发工作进展。报告按日期分类存储，便于后续查阅和总结。

## 文件结构

```
dayly-report/
├── README.md          # 项目说明
├── 2026/
│   ├── 02-23.md      # 2026年2月23日报告
│   └── ...
└── ...
```

## 如何使用

1. 查看指定日期的报告：`[日期].md`
2. 生成新的日报：按照现有格式创建新的日期文件
3. 总结工作：通过历史报告总结工作进展

## 格式规范

所有日报文件都遵循以下Markdown格式：

```markdown
# 开发日报（YYYY-MM-DD）

## 今日摘要

(一句话总结主要产出)

## 详细变更

### 新增功能

- [模块名] 详情...

### 问题修复

- [模块名] 详情
```
